# Plasticity Project

Using neuroplasticity to understand how minds, markets, and systems adapt under stress.

## Core Idea
Change is not constant. It emerges in windows where systems become more susceptible to influence.

This project tracks:
- Signals (real-world events & patterns)
- Ideas (emerging hypotheses)
- Framework (structured model of plasticity)
- Essays (refined outputs)

## Philosophy
Constrain noise. Track signals. Let patterns emerge before forcing conclusions.
